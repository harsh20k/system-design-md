# System-Design.md

A 2-month daily System Design interview practice log, building toward the AWS Solutions Architect Professional (SAP-C02) certification. One question is tackled per day, rotating through scalability, high availability/DR, microservices, databases, messaging, networking, security, cost optimization, migration, multi-region, serverless, and storage topics — starting with fundamentals and escalating to full AWS-specific mock scenarios.

## Overview

- **Cadence:** 1 question/day, ~60 days
- **Tracking:** each day's question is logged as an issue in the Linear project `System-Design.md`
- **Goal:** be exam-ready for AWS Solutions Architect Professional (SAP-C02) by the end of the 2 months
